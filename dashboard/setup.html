Conecta LinkedIn, la IA y el Worker de Cloudflare. Solo necesitas hacerlo una vez.

🔌 Estado de las conexiones

⏳ Comprobando conexiones…
1 API del Worker (Cloudflare)
La URL de tu Cloudflare Worker desplegado. Ejemplo: https://liberfy-linkedin.TU_SUBDOMAIN.workers.dev
Se guarda solo en tu navegador (localStorage). Nunca se sube a ningún servidor.
El token Bearer que configuraste como secret en el Worker.
2 GitHub (para triggers manuales)
Opcional. Permite lanzar los scrapers manualmente desde el dashboard.
Necesita permisos de workflow. Crea uno en GitHub → Settings → Developer settings → Tokens.
3 LinkedIn OAuth
Conecta tu cuenta de LinkedIn para que el sistema pueda publicar posts en tu nombre. Este proceso se hace desde el Worker de Cloudflare.
1️⃣ Ve a developer.linkedin.com/apps/new y crea una app
2️⃣ En "Products", activa Share on LinkedIn y Sign In with LinkedIn using OpenID Connect
3️⃣ En "Auth", añade el redirect URI de tu Worker: [introduce tu Worker URL primero]
4️⃣ Introduce las credenciales en los secretos de tu Cloudflare Worker (LINKEDIN_CLIENT_ID, LINKEDIN_CLIENT_SECRET)
5️⃣ Haz clic en el botón de autorizar para vincular tu cuenta
Introduce primero la URL del Worker para habilitar este paso.
